# Anxiety Time Machine

A cross-generational anxiety sharing platform where users can share and browse anxieties from different age groups.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open http://localhost:3000 in your browser

## Features

- User registration and login
- Share personal anxieties
- Browse anxieties by age range
- Filter by anxiety type (Academic/Career)

## Tech Stack

- React 18
- Tailwind CSS
- Lucide React (icons)
